## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/akash-portfolio.git
cd akash-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:3000`

### Build for Production

```bash
npm run build
```

The build output will be in the `dist` folder.

## Customization

### Update Personal Information

1. Edit the content in each component file to match your information
2. Update social links in [Contact.jsx](src/components/Contact.jsx)
3. Modify skills in [Skills.jsx](src/components/Skills.jsx)
4. Update work experience in [Experience.jsx](src/components/Experience.jsx)
5. Modify education details in [Education.jsx](src/components/Education.jsx)

### Styling

All component styles are in their respective CSS files. The main global styles are in [index.css](src/index.css).
